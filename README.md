🎓 Student ERP System

A production-style Student ERP (Enterprise Resource Planning) backend built with Java, Spring Boot, Spring Security, JWT Authentication, Hibernate/JPA, MySQL, and MapStruct. The project follows a clean layered architecture and RESTful API design to manage academic operations in schools and colleges.

🚀 Features
🔐 JWT Authentication & Authorization
👥 Role-Based Access Control (Admin, Teacher, Student)
👤 User Registration & Login
🎓 Student Management (CRUD)
👨‍🏫 Teacher Management (CRUD)
📚 Subject Management (CRUD)
📝 Enrollment Management
📅 Attendance Management
📊 Marks Management
🏆 Automatic Grade Calculation
📄 Student Transcript Generation
🎯 GPA Calculation
✅ Request Validation
⚠️ Global Exception Handling
🔄 DTO Mapping using MapStruct
